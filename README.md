# Transloco Démo

```ng serve````

## Démo

1. Ajout de la bibilothèque Transloco (ng add @jsverse/transloco)
2. Configuration dans app.config.ts
3. Configuration du TranslocoHttpLoader et des fichiers de traduction (public/i18n/*)
3. Imports dans AppComponent
4. Utilisation de la directive dans le template
6. Import de locale dans main.ts
5. Utilisation des pipe dans le template (et import)
